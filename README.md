# ApplyNow - Smart Admissions Application

This repository contains everything you need to run the ApplyNow application locally. Developed by [@asif_naija](https://www.instagram.com/asif_naija/).

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```
   npm install
   ```
2. Set the `GEMINI_API_KEY` in `.env.local` to your Gemini API key
3. Run the app:
   ```
   npm run dev
   ```

## Features

- AI-powered admission chance prediction
- Interactive application dashboard
- Real-time analysis using Google Gemini AI
- Responsive design for all devices

## Tech Stack

- React 19
- TypeScript
- Vite 6
- Tailwind CSS
- Google Gemini AI
- React Hook Form
- Zod validation

## Deployment

To build the application for production:
```
npm run build
```

To preview the production build locally:
```
npm run preview
```
